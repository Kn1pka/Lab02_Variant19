# Лабораторная работа №2: Разработка приложений Windows Forms
## Вариант 19: Числа Армстронга

Графическое приложение на C# (Windows Forms), предназначенное для проверки, генерации и визуализации свойств чисел Армстронга.

📋 Описание
Программа реализует интерфейс с тремя независимыми задачами:

1. **Проверка на число Армстронга** — определяет, является ли введённое число числом Армстронга (сумма цифр в степени количества цифр равна самому числу).
2. **Генератор чисел** — находит и выводит в список (`ListBox`) все числа Армстронга в диапазоне `[1, N]`.
3. **Визуализация проверки** — отображает в `PictureBox` графическую схему процесса: каждая цифра числа, её степень, промежуточные суммы, итоговый результат. Используются блоки, стрелки и текстовые подписи.

⚙️ Особенности и требования
- Входные данные: целые неотрицательные числа.
- Для задачи №2 значение `N` должно быть `≥ 1`.
- Приложение включает строгую валидацию ввода: защита от букв, пустых полей, отрицательных чисел и нуля там, где это не допускается.
- Интерфейс создаётся программно (динамическая генерация `GroupBox`, `TextBox`, `ListBox`, `PictureBox`), что исключает конфликты с файлом дизайнера.
- Отрисовка схемы выполняется в событии `Paint` с использованием класса `Graphics` (методы `FillRectangle`, `DrawString`, `DrawLine` и др.).
- Дробные числа не используются, ввод строго целочисленный.

🚀 Запуск проекта
**Через Visual Studio**
1. Откройте файл решения `Lab02_Variant19.sln`
2. Убедитесь, что выбрана конфигурация `Debug` / `Any CPU`
3. Нажмите `F5` для запуска с отладкой или `Ctrl+F5` без отладки
4. Форма автоматически сгенерирует все элементы управления и будет готова к работе

📦 Структура проекта
- `Form1.cs` — основной код формы, логика задач, обработчики событий `Click` и `Paint`
- `Form1.Designer.cs` — минимальный каркас дизайнера (интерфейс инициализируется в коде)
- `Program.cs` — точка входа в приложение
