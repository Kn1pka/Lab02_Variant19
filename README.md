 Описание
Данный проект представляет собой приложение на C# с использованием технологии Windows Forms. Программа предназначена для работы с числами Армстронга (нарциссическими числами) и выполняет три основные функции.

> Число Армстронга — это число, сумма цифр которого, возведенных в степень, равную количеству цифр в числе, равна самому числу.
> Пример: 153 = 1^3 + 5^3 + 3^3

Функционал приложения

Задача 1. Проверка на число Армстронга
Пользователь вводит целое число. Приложение проверяет, является ли оно числом Армстронга, и выводит соответствующее сообщение.
Входные данные: Целое число.
Выходные данные: Логический ответ (Является / Не является).

Задача 2. Генератор чисел Армстронга
Пользователь задает верхнюю границу диапазона $N$. Приложение находит и выводит в список (`ListBox`) все числа Армстронга в диапазоне $[1, N]$.
Входные данные: Целое число $N > 0$.
Выходные данные: Список найденных чисел.

 Задача 3. Визуализация проверки
Для введенного числа приложение строит графическую схему в элементе `PictureBox`.
 Отображает каждую цифру числа в отдельном блоке.
 Показывает степень, в которую возводится цифра.
 Вычисляет сумму степеней.
 Сравнивает сумму с исходным числом и выдает результат.
 Графические элементы: Блоки, стрелки, текстовые подписи.

  Технологии

  Установка и Запуск
1. Откройте файл решения `Lab02_Variant19.sln` в среде Visual Studio.
2. В меню выберите Сборка (Build) -> Собрать решение (Build Solution) или нажмите `Ctrl + Shift + B`.
3. Для запуска нажмите клавишу `F5` или кнопку Запуск (Start).
